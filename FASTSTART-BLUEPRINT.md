## Phase 1: The 6-Phase Launch System

This is the sequence. Follow it in order. Don't skip. Don't reorder.

### Phase 1: Idea (5 minutes)
**What you're doing:** Picking what to build

**The Framework:**
- Is it solvable in under 2 hours?
- Can you explain it in one sentence?
- Would you pay $27 for this if someone else made it?

If yes to all three â†’ move forward.
If no to any â†’ pick a different idea.

**Common mistakes:**
- Picking ideas that are too complex
- Waiting for the "perfect" idea
- Overthinking market research

**The fix:**
Pick something small. You can always expand later.

---

**What you're doing:** Writing the sales page

**The Framework:**

### Phase 3: Product (7 minutes)
**What you're doing:** Creating what they get

**Three Proven Models:**

#### Model 1: The Guide
- Write a step-by-step breakdown
- Include examples and screenshots
- Format: PDF or Notion doc
- Time to create: 30-60 minutes (do this AFTER you have buyers)

#### Model 2: Template Pack
- Notion templates, spreadsheet templates, prompt templates
- Package 3-5 templates together
- Add instructions doc
- Time to create: 45 minutes

#### Model 3: Micro-Tool
- Simple calculator, generator, or automation
- Use no-code tools (Bubble, Softr, Airtable)
- Keep it focused on ONE thing
- Time to create: 1-2 hours

**For the FastStart launch:**
I promised the guide but created the sales page first. Why? Because you need buyers before you need the product.

Launch first. Build second.

---

### Phase 4: Payment (5 minutes)
**What you're doing:** Connecting Stripe

**Step-by-step:**
1. Go to stripe.com (create account if needed)
2. Click "Payment Links"
3. Click "Create payment link"
4. Enter:
   - Product name
   - Price ($27 recommended for first launch)
   - Description (one sentence)
5. Copy the link
6. Add to your sales page CTA button

**That's it.** Don't overcomplicate this.

**No Stripe account yet?**
Use Gumroad or Lemon Squeezy. Same process. Takes 5 minutes.

## Phase 4: Pre-Built Prompt Templates

### Sales Copy Generation Prompt
```
I'm launching a micro-business. Help me write sales copy.

Product: [DESCRIBE IN ONE SENTENCE]
Target buyer: [WHO IS THIS FOR]
Main benefit: [WHAT PROBLEM IT SOLVES]
Price: $[AMOUNT]

Use this structure:
1. Hook (outcome + mechanism)
2. Problem (why current solutions fail)
3. Mechanism (how mine solves it)
4. Deliverables (what they get)
5. Objections (address 3 common ones)
6. Guarantee
7. CTA

Write it conversationally. No hype words. Be specific.
```

---

### Product Outline Prompt
```
I need to structure a [GUIDE/TEMPLATE PACK/TOOL].

Topic: [WHAT IT'S ABOUT]
Goal: Help people [DESIRED OUTCOME] in [TIMEFRAME]

Create an outline with:
- 5-7 main sections
- 3-5 subsections per main section
- Concrete deliverables (what files/resources to include)

Keep it simple. Actionable only.
```

## Phase 6: The 3 Proven Micro-Business Models

Pick one. Launch this week.

### Model 1: The Process Guide
**What it is:** Step-by-step breakdown of how you do something

**Examples:**
- "How I Get 10K Views on Every Tweet"
- "My $5K/Month Freelance Workflow"
- "The 1-Hour Product Launch System" (this guide!)

**How to create:**
1. Pick a process you've done 10+ times
2. Document every step
3. Add screenshots/examples
4. Export as PDF

**Time to create:** 2-3 hours  
**Price point:** $19-49

---

### Model 2: Template Pack
**What it is:** 3-5 ready-to-use templates

**Examples:**
- "5 Notion Templates for Solopreneurs"
- "Cold Email Templates That Convert"
- "Social Media Content Calendar Template"

**How to create:**
1. Take templates you already use
2. Clean them up
3. Remove personal info
4. Add instructions doc
5. Package together

**Time to create:** 1-2 hours  
**Price point:** $9-29

---

### Model 3: Micro-Tool
**What it is:** Simple calculator/generator/automation

**Examples:**
- "Freelance Rate Calculator"
- "Social Media Caption Generator"
- "ROI Calculator for Ads"

**How to create:**
1. Use no-code tool (Softr, Bubble, Airtable)
2. Build ONE specific function
3. Make it instantly usable
4. Share link

**Time to create:** 2-4 hours  
**Price point:** $7-19 (or free + upsell)

## Appendix B: Common Mistakes

### Mistake 1: Building Before Selling
**The trap:** Spend weeks building, launch to crickets.

**The fix:** Sell first. Promise delivery in 7 days. Build after first sale.

---

### Mistake 2: Over-Designing
**The trap:** Tweaking colors/fonts/layout for days.

**The fix:** Pick a template. Don't touch it. Focus on copy.

---

### Mistake 3: Waiting for "Perfect"
**The trap:** Never launching because "it's not ready."

**The fix:** Ship version 0.1. Update after launch.

---

### Mistake 4: Overthinking Price
**The trap:** Analyzing competitor pricing for hours.

**The fix:** Start at $27. Adjust based on sales data.

---

### Mistake 5: No Guarantee
**The trap:** People hesitate because of risk.

**The fix:** Add 7-day or 30-day refund. State it clearly.
