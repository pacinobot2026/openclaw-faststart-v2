# GALLERY VARIETY FIX NEEDED

## Problem
All 33 designs look identical:
- Same padding (60px 40px)
- Same headline size (36px)
- Same subheadline size (16px)
- Same min-height (420px)
- Same everything

## SOLUTION - Add 3 Template Sizes

### SIZE 1: FULL (8 designs)
- padding: 80px 50px
- min-height: 480px
- headline: 42-44px
- subheadline: 18px
- Spacious, airy

Examples: Tokyo Midnight, Desert Oasis, Royal Purple, Cyber Miami, Sunset Blaze, Midnight Gold, Ultra Gradient, Deep Space

### SIZE 2: MEDIUM (15 designs)  
- padding: 60px 40px
- min-height: 420px
- headline: 38-40px
- subheadline: 16px
- Balanced

Examples: Nordic Frost, Electric Blue, Arctic Frost, Emerald Dark, Crimson Power, Carbon Fiber, Most others

### SIZE 3: COMPACT (10 designs)
- padding: 40px 30px
- min-height: 360px
- headline: 32-34px
- subheadline: 15px
- Tight, focused

Examples: Traditional Marketing, Social Media, Glassmorphism, Cotton Candy, Champagne, Espresso, Berry Smoothie

## ALSO VARY
- Headline fonts (some bigger, some condensed)
- Button sizes (some full, some auto width)
- Spacing between elements

## CURRENT STATE
Everything uses:
.template { padding: 60px 40px; min-height: 420px; }
.headline { font-size: 36px; }
.subheadline { font-size: 16px; }

## FIX NEEDED
Add specific classes:
.template-full { padding: 80px 50px; min-height: 480px; }
.template-compact { padding: 40px 30px; min-height: 360px; }

Then assign sizes to designs for variety.
