# Promotions Directory

Scheduled and planned promotions for email broadcasts, social media, and campaigns.

## Structure

```
promotions/
├── README.md          # This file
├── scheduled/         # Confirmed promotions with dates
├── ideas/             # Potential promotions to consider
└── templates/         # Reusable promotion templates
```

## Adding a Promotion

Create a markdown file with:
- **Subject/Headline**
- **Target Audience** (tags, segments)
- **Scheduled Date/Time**
- **Content/Copy**
- **Links/CTAs**
- **Status** (draft/scheduled/sent)

## Quick Reference

| Status | Meaning |
|--------|---------|
| 📝 Draft | Still being written |
| ✅ Ready | Approved, waiting to schedule |
| 📅 Scheduled | Date confirmed |
| ✉️ Sent | Completed |
