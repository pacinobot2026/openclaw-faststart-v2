# VSL Skill - Quick Reference

## One-Line Commands

```
Create VSL for [sales page URL]
```

```
Generate video sales letter from operator-version page
```

```
Build VSL: [URL] | Voice: Pacino | Length: 8min
```

---

## Workflow

**You say:** "Create VSL for [page]"

**I do:**
1. ✅ Read sales page
2. ✅ Extract core elements
3. ✅ Write script (1,200+ words)
4. ✅ Generate voice (ElevenLabs)
5. ✅ Create slides
6. ✅ Render video (FFmpeg)
7. ✅ Deliver MP4

**You get:** Finished video file

## Quick Customization

**Change voice tone:**
```
VSL for [URL] - use authoritative/calm/excited tone
```

**Change length:**
```
VSL for [URL] - make it 5 minutes
```

**Emphasize specific points:**
```
VSL for [URL] - focus on proof and urgency
```

- `slides/` folder - Visual slides
- `final-vsl.mp4` - Rendered video

All saved to project folder or workspace.
