## Usage Notes

**When creating sales pages, reference these for:**

1. **Layout Structure** - Use #1 as template guide
2. **Celebrity/Authority** - See #2, #8 for endorsement examples  
3. **Urgency/Scarcity** - #4, #6 use countdown timers effectively
4. **Social Proof** - #4, #8 show logo walls and student counts
5. **Problem-Aware Copy** - #5, #7 address pain points directly
6. **Video Integration** - #3, #6, #8, #9 use video as primary CTA
7. **Pricing Display** - #2, #4, #5, #8 show different pricing strategies
8. **Color Psychology** - #6 (black = premium), #9 (blue = trust), #10 (warm = approachable)
9. **Headlines** - Study the variety: question-based, benefit-driven, problem-focused
10. **Free Lead Magnets** - #10 shows how to position free training
