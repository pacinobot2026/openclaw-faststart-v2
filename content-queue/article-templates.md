## Template: EVENTS 🎉

**Alert:** EVENTS (Yellow/Orange background)
**Word Count:** 300-400
**Tone:** Excited, informative, community-focused

### Structure:
1. **Opening** - What's happening, when, where (1-2 sentences)
2. **Details** - What to expect, activities, features (2-3 paragraphs)
3. **Practical info** - Tickets, parking, times, age restrictions
4. **Closing** - Why this matters to community, call to action

### Example Opening:
```
<p><strong>[Event Name]</strong> returns to <u>[Location]</u> from <strong>[Date]</strong>, inviting the community to [brief description].</p>
<p>&nbsp;</p>
```

### Keywords Pattern:
- Event name
- Location (Summerlin, Downtown Summerlin, etc.)
- Type of event (Festival, Parade, Market, etc.)

## Formatting Rules (ALL Templates)

### Paragraph Spacing (CRITICAL!)
```html
<p>First sentence here.</p>
<p>&nbsp;</p>
<p>Second sentence here.</p>
<p>&nbsp;</p>
```

### Strategic Bolding
- **Location names:** Summerlin, Las Vegas, Nevada
- **Business/Organization names:** First mention
- **Key people:** First mention
- **Important dates/times**

### Avoid Bolding
- Every keyword (looks spammy)
- Common words
- Entire sentences

### Keywords (3 minimum)
1. Primary subject
2. Location
3. Topic/category term
