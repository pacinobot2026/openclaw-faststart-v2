## 2:58 AM - STRIPE + VERCEL INTEGRATION (CRITICAL LESSON)

**Problem:** Chad's Stripe API key is an **Organization key** (`sk_org_live_...`). These require the Account ID for ALL API calls.

**Chad's Stripe Account ID:** `acct_1T4AreCDxYH1XF8F`

**Solution:** Add `stripeAccount` option to every Stripe API call:
```javascript
stripe.checkout.sessions.create({...}, {
  stripeAccount: 'acct_1T4AreCDxYH1XF8F'
});
```

**Vercel env vars needed for any Stripe project:**
1. `STRIPE_SECRET_KEY` - The API key
2. `STRIPE_ACCOUNT_ID` - `acct_1T4AreCDxYH1XF8F`
