## Email Template Format (UPDATED per Chad's screenshot)

**OLD FORMAT (numbered lists) - DON'T USE:**
```
1. Topic 1
2. Topic 2
3. Topic 3
```

**NEW FORMAT (emoji bullets) - USE THIS:**
```
🚀 Topic 1 - brief description

📧 Topic 2 - brief description

✨ Topic 3 - brief description
```

**Key rules:**
- Link at TOP (after opening line)
- Link at BOTTOM (before sign-off)
- "What we're covering:" with emoji bullets
- Each bullet = emoji + topic + brief description
- Keep it punchy and exciting
- NO numbered lists - use emoji bullets instead
