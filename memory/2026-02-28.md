## 2026-02-28 23:10:40 - WISHLIST PROTOCOL: Clean URLs

**Rule:** Always remove UTM parameters before saving URLs to wishlist
- Strip: ?fbclid=, &utm_source=, &utm_medium=, &utm_campaign=, etc.
- Save clean base URL only
- Example: https://example.com/product?fbclid=xyz ? https://example.com/product

**Applies to:** All /wishlist commands


## 2026-02-28 23:33:27 - URL METADATA RULE (UNIVERSAL)

**ALWAYS extract for ANY URL:**
1. Meta title (og:title or <title>)
2. Meta description (og:description or meta description)
3. Meta image (og:image or twitter:image)

**This applies to:**
- Wishlist items
- Bookmarks
- Article ideas
- ANY feature that saves URLs

**Never save a URL without attempting to fetch all three pieces of metadata.**
