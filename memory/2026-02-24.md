## 🎨 OpenClaw FastStart V2 - Typography & Layout Refinement

### Session Summary
Worked with Chad on iterating the FastStart Blueprint sales page design. Multiple rapid iterations on typography hierarchy, spacing, and layout.

### Key Design Lessons Learned

#### Typography Hierarchy Rules (CRITICAL)
1. **Order matters** - Subheadline → Main Headline → Post-headline
2. **Size contrast is everything** - Main headline MUST be noticeably bigger than sub/post
3. **Centered alignment** - Keep all headlines centered for maximum impact
4. **Too big is real** - text-9xl was too massive, text-6xl hit the sweet spot
5. **Borders can distract** - Removed gradient border from main headline - cleaner without it

**Final working sizes:**
- Subheadline: `text-2xl md:text-3xl lg:text-4xl`
- Main Headline: `text-3xl md:text-5xl lg:text-6xl`
- Post-headline: `text-xl md:text-2xl lg:text-2xl`

#### Layout Lessons
1. **Split layout failed** - Text left/video right felt unbalanced for this message
2. **Video placement** - Below post-headline works better than side-by-side
3. **Spacing at top** - `py-20` was too much, `py-12` felt right
4. **Max-width matters** - Keep headlines in `max-w-5xl` or `max-w-6xl` for readability

#### The Iteration Process (How Chad Works)
- Start with concept → iterate FAST
- Don't ask "should I?" - just deploy and show
- Typography is trial-and-error until it feels right
- "Fix it revert back" means go smaller/simpler
- When he says "perfect but..." - you're 95% there

## 🎯 3-Part Headline Doctrine (2:07 AM)

**Saved to:** `skills/copywriting/OPENCLAW-HERO-3PART-HEADLINE-DOCTRINE.md`

**Structure:**
1. **Pattern Interrupt** (3-7 words, short & sharp, stop the scroll)
2. **Mechanism Reveal** (clarity, explains how)
3. **Outcome + Identity Shift** (what it means for reader)

**Critical rule:** Each line must answer a DIFFERENT subconscious question:
- Line 1: "Why should I care?"
- Line 2: "What is this?"
- Line 3: "What does this do for me?"

**Example:**
- Part 1: "Humans Are the Bottleneck."
- Part 2: "An OpenClaw agent built this entire offer in 37 minutes."
- Part 3: "Now you can launch faster than 90% of creators."

**5 Strategic Angles:**
A) Speed dominance  
B) Fear of falling behind  
C) Operator identity  
D) AI inevitability  
E) Clean premium execution  

**Next step:** Lock in ONE angle for consistent brand identity across all pages.
