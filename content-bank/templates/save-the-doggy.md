## Usage Instructions

1. **Use real Las Vegas rescue organizations:**
   - The Animal Foundation (main shelter)
   - Nevada SPCA
   - Hearts Alive Village
   - Any local rescue Chad knows

2. **Emotional but authentic:**
   - Use real dog stories when possible
   - Keep it hopeful, not guilt-trippy
   - Focus on positive outcomes

3. **Always include clear CTAs:**
   - How to adopt
   - Where to visit
   - Application links
   - Contact info

4. **Photo suggestions:**
   - If no real dog photos available, use stock rescue imagery
   - Before/after transformations
   - Happy adoption day photos

**SEO Strategy:**
- Bold dog names
- Bold breed names
- Bold rescue organization names
- Focus on Las Vegas local rescue scene
